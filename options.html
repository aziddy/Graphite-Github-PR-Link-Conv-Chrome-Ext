Keyboard shortcuts

  • Convert PR link … (default)
  • Scroll to top of PR … (default)

Are you sure? Chrome doesn't let extensions reset their own shortcuts — clicking Yes opens Chrome's shortcut editor in a new tab. There, find PR Link Converter, click the pencil icon next to each command, and press:

  • Convert PR link: ⌘⇧L (macOS) or Ctrl+Shift+L (Windows/Linux)
  • Scroll to top of PR: ⌘⇧U (macOS) or Ctrl+Shift+U (Windows/Linux)
